# 2D-Navier-Stokes-FEM-solver-with-GLS-stabilization-in-C-
A high-fidelity 2D Navier-Stokes solver written in C++. Features an unsteady, incompressible rotational flow in Lid-deriven cavity utilizing Galerkin/Least-Squares (GLS) stabilization and low-dissipation schemes for accurate viscous effect modeling.
